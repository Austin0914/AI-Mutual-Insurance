# adr/ — 架構決策紀錄

記錄**重大且不易回頭**的技術決策，保留當時的脈絡與取捨，避免日後反覆爭論。

## 規則

- 一個決策一個檔案，編號遞增：`NNNN-標題.md`。
- 決策一旦被取代，不刪除舊檔，改將其狀態標為 `Superseded by ADR-XXXX`。
- 範本見 [0000-template.md](0000-template.md)。

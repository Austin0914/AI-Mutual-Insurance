# 系統總覽

> 草稿。本檔提供全局心智模型；機制定義以 [白皮書](../BluePrint/AI共險基金白皮書.md) 為準。

## 鏈上 / 鏈下分界

協議的核心原則是「**鏈管錢，不管真相**」：

- **鏈上（contracts/）**：資金託管、注資、鎖倉、commit-reveal 投票紀錄、加權統計、罰沒、分層結算。
- **鏈下（backend/）**：證據暫存與摘要、事件索引、KYC/KYB 流程、爭議案件的非機密中介資料、通知。
- **介面（frontend/）**：參與者操作（注資、索賠、投票 commit/reveal、查詢公信力與基金狀態）。

## 參與者 → package 對照

| 參與者（見白皮書 §5）  | 主要互動介面                                            |
| ---------------------- | ------------------------------------------------------- |
| AI 提供商 / 使用者公司 | frontend（注資、索賠）                                  |
| 鑑定者                 | frontend（commit / reveal 投票）、backend（取證據摘要） |
| KYC Gatekeeper         | backend（准入審核流程）                                 |
| 智慧合約               | contracts（自動執行）                                   |

## 三個 package 的職責邊界

- **contracts/**：協議的唯一權威執行層。任何「錢怎麼動、規則怎麼跑」最終都由合約決定。
- **backend/**：輔助層。**不得**成為資金或判定的權威來源；只做索引、暫存、合規流程與 API。
- **frontend/**：純展示與互動，所有狀態最終以鏈上為準。
